# Enlightiam GitHub Pages

This is the public Enlightiam website for GitHub Pages.

- `index.html` is the homepage.
- `privacy.html` and `terms.html` are the public legal pages.
- `assets/` contains the logo and showcase image.
- `media/` contains temporary public carousel images that Buffer can retrieve.
- The **Open creator** button goes to the private Enlightiam Creator workspace, where uploads and seven-day cleanup continue to run securely.

## Publish

Upload the **contents of this folder** to the root of `AlemuHub/enlightiam-media`. Then open **Settings → Pages**, choose **Deploy from a branch**, select `main` and `/(root)`, and save.

Your Pages URL will be:

`https://alemuhub.github.io/enlightiam-media/`
